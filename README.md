# Formix 使用文档

## 一、软件简介

Formix 是一个基于 FFmpeg 的桌面多媒体处理工具，当前包含以下功能：

1. 视频格式转换
2. 音频格式转换
3. 图片格式转换
4. M3U8 下载与转封装
5. 音视频分离
6. 音视频合成
7. GPU 编码、背景设置、每日壁纸、在线更新

## 二、运行前准备

### 1. 安装 Python 依赖

如果你当前就在 `format_factory` 目录中，执行：

```bash
pip install -r requirements.txt
```

### 2. 准备 FFmpeg

本项目默认从软件目录内读取 FFmpeg，不会自动使用系统环境变量中的 FFmpeg。

请确认以下文件存在：

`format_factory/FFmpeg/bin/ffmpeg.exe`

推荐同时放入：

`format_factory/FFmpeg/bin/ffprobe.exe`

说明：

- `ffmpeg.exe` 是必须文件，缺失时软件无法启动。
- `ffprobe.exe` 不是强制要求，但建议提供。它用于识别输入文件信息、显示格式、优化部分下拉选项。

### 3. 启动方式

建议在 `format_factory` 的上一级目录执行：

```bash
python -m format_factory.main
```

如果你当前已经进入 `format_factory` 目录，可先返回上一级目录再执行。

## 三、主界面说明

软件主界面包含 6 个分页：

1. 视频
2. 音频
3. 图片
4. M3U8
5. 音视频
6. 设置

大多数转换页的操作流程一致：

1. 添加输入文件
2. 选择输出格式
3. 选择输出目录
4. 根据需要调整预设或参数
5. 点击“开始”
6. 在日志区查看进度、警告和结果

通用操作：

- 支持拖拽文件到列表中
- 支持批量处理
- 支持复制日志和清空日志
- 支持取消当前任务与后续排队任务

## 四、视频转换

### 功能

- 常见视频格式互转
- 可选分辨率缩放
- 可选预设与自定义 FFmpeg 参数
- 支持转为 HLS 播放列表

### 支持的输出格式

- `mp4`
- `m4a`
- `mkv`
- `avi`
- `mov`
- `webm`
- `flv`
- `gif`
- `m3u8`

### 基本步骤

1. 点击“添加文件”或“添加文件夹”
2. 在“格式”中选择目标格式
3. 如果是普通视频，可额外选择分辨率
4. 在“预设 / 参数”中选择默认预设、高质量、快速压缩等
5. 选择输出目录
6. 点击“开始转换”

### 说明

- 转 `gif` 时更适合短视频片段。
- 转 `m3u8` 时，软件会在输出目录下为每个文件创建一个 `原文件名_segments` 子目录，里面保存 `.m3u8` 文件和 `.ts` 切片。
- 如果选择“自定义”参数，请确保自己熟悉 FFmpeg 语法。

## 五、音频转换

### 功能

- 常见音频格式互转
- 支持从带封面的媒体中提取音频
- 支持 `.ncm` 文件解密后再转换

### 支持的输出格式

- `mp3`
- `m4a`
- `aac`
- `wav`
- `flac`
- `ogg`
- `opus`

### 基本步骤

1. 添加音频文件
2. 选择输出格式
3. 选择输出目录
4. 选择预设或自定义参数
5. 点击“开始转换”

### `.ncm` 说明

- 软件会先解密 `.ncm`，再按你的目标格式继续处理。
- 解密过程中会临时生成缓存文件到 `ncm_cache` 目录。
- 任务完成后软件会自动尝试清理缓存。

## 六、图片转换

### 功能

- 常见图片格式互转
- 提供若干缩放与质量预设

### 支持的输出格式

- `jpg`
- `png`
- `webp`
- `bmp`
- `tiff`
- `ico`

### 基本步骤

1. 添加图片文件
2. 选择目标格式
3. 选择输出目录
4. 根据需要选择质量或缩放预设
5. 点击“开始转换”

### 说明

- 转 `ico` 时适合生成图标文件。
- 如果只想改尺寸，可优先选择带缩放的预设。

## 七、M3U8 下载与转封装

### 功能

- 输入网络 M3U8 地址进行下载
- 也支持加载本地 `.m3u8` 文件
- 可输出为普通视频文件，也可继续输出为 M3U8 结构

### 支持的输入方式

- 在线 `http://` 或 `https://` 链接
- 本地 `.m3u8` 文件

### 支持的输出格式

- `mp4`
- `mkv`
- `avi`
- `mov`
- `webm`

### 使用步骤

1. 在输入框中粘贴 M3U8 链接，或点击按钮选择本地 `.m3u8`
2. 选择输出格式
3. 选择输出目录
4. 点击“开始下载/转换”

### 说明

- 若输出为视频格式，软件会尝试直接封装或转换。
- 若后续版本开放 M3U8 结构输出，切片文件会放在单独子目录中。
- 某些加密流、失效链接、需要登录态的链接可能无法直接下载。

## 八、音视频页

音视频页包含两个子页：

1. 分离音视频
2. 合成音视频

### 1. 分离音视频

#### 功能

- 从视频中提取纯音频
- 从视频中提取纯视频
- 同时提取音频和视频

#### 使用步骤

1. 添加一个或多个视频文件
2. 选择提取模式
3. 选择音频输出格式和或视频输出格式
4. 选择输出目录
5. 点击开始

#### 输出命名规则

- 提取音频：`原文件名_audio.扩展名`
- 提取视频：`原文件名_video.扩展名`

#### 建议

- 提取纯视频时优先选择 `mp4`、`mkv`、`mov`、`avi`、`webm`、`flv`
- 提取纯音频时按播放器兼容性优先选择 `mp3` 或 `m4a`

### 2. 合成音视频

#### 功能

- 将一个音频文件批量合成到多个视频文件中

#### 使用步骤

1. 添加一个或多个视频文件
2. 选择一个音频文件
3. 选择输出格式
4. 选择音频处理方式
5. 选择输出目录
6. 点击开始

#### 音频处理方式说明

- 复制音频流：速度快、无损，推荐优先尝试
- 转为 M4A(AAC)：兼容性较好
- 转为 AAC 192k：适合常见视频容器
- 转为 MP3 192k：兼容旧设备
- 转为 FLAC：无损但体积较大
- 转为 Opus 96k：体积小但兼容性不如 AAC/MP3

#### 输出命名规则

- `原视频名_merged.扩展名`

## 九、设置页

设置页主要包含以下内容：

1. 外观
2. GPU 设置
3. 关于
4. 软件更新

### 1. 外观

可设置内容：

- 亮色 / 暗色主题
- 背景图片
- 模糊强度
- 遮罩透明度
- 每日壁纸

说明：

- 选择背景图后，软件会根据图片整体颜色自动调整部分界面风格。
- 模糊和遮罩设置主要影响背景显示效果。
- 清除背景时，会同时关闭每日壁纸功能。

### 2. 每日壁纸

#### 功能

- 从在线接口获取当日壁纸
- 自动缓存到本地
- 可手动刷新

#### 说明

- 开启后，软件启动时会优先读取当天缓存。
- 每天零点会自动尝试刷新下一张壁纸。
- 如果你手动更换背景图，每日壁纸会自动关闭。

### 3. GPU 设置

支持选项：

- NVIDIA NVENC
- AMD AMF
- Intel Quick Sync
- 不使用 GPU

说明：

- GPU 加速主要用于视频编码。
- 只有 FFmpeg 编译时包含对应编码器，GPU 加速才会生效。
- 遇到不支持 GPU 的预设时，软件会自动回退到 CPU。
- 一般只对 H.264 和 H.265/HEVC 编码有明显帮助。

### 4. 软件更新

#### 功能

- 手动检查新版本
- 显示更新公告
- 下载更新包并准备安装

#### 说明

- 检查更新需要网络连接。
- 如果发现新版本，界面会显示版本号、发布日期和更新说明。

## 十、日志说明

每个处理页底部都有日志区域，用于显示：

1. 已添加文件
2. 当前任务队列
3. FFmpeg 命令
4. 编码器信息
5. 实时进度
6. 警告与错误
7. 成功或失败结果

常见日志类型：

- `info`：普通信息
- `meta`：输入输出信息、格式识别信息
- `cmd`：执行的 FFmpeg 命令
- `encoder`：编码器与 GPU 相关信息
- `warning`：警告
- `error`：错误
- `success`：成功

## 十一、常见问题

### 1. 软件启动后提示找不到 FFmpeg

请检查：

- `FFmpeg/bin/ffmpeg.exe` 是否存在
- 文件路径是否放在 `format_factory` 目录内部

### 2. 文件可以转换但看不到格式识别信息

通常是 `ffprobe.exe` 缺失导致。请补充：

`FFmpeg/bin/ffprobe.exe`

### 3. GPU 选项已选择，但仍然没有加速

可能原因：

- 当前预设不支持 GPU
- 当前 FFmpeg 没有编译对应 GPU 编码器
- 目标格式本身不适合 GPU 编码

### 4. M3U8 无法下载

可能原因：

- 链接失效
- 流媒体需要登录或鉴权
- 服务器限制直链访问
- M3U8 包含额外加密

### 5. 转换失败如何排查

建议按下面顺序检查：

1. 查看日志区的最后几条错误信息
2. 检查输入文件是否损坏
3. 更换目标格式重试
4. 关闭 GPU 后重试
5. 去掉自定义参数后重试

## 十二、使用建议

1. 新手建议先使用默认预设
2. 需要兼容性时优先选择 `mp4 + aac`
3. 需要高压缩率时可尝试 `H.265`
4. 需要无损音频时优先选择 `flac` 或 `wav`
5. 需要网页播放时优先选择 `mp4` 或 `webm`
6. 大批量处理前，建议先拿 1 个文件测试参数是否符合预期
